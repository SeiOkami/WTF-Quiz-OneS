title: 'Новый Цвет(-1,-255, -256)'
question: |
    ```bsl
    Ц = Новый Цвет(-1,-255, -256);
    Сообщить(Ц);
    ```
options:
    - -1, -255, -256
    - -1, -255, 0
    - 1, 255, 0
    - 1, 255, 1
    - 255, 1, 0
    - 255, 0, 1
    - Исключение
correct: 4
explanation: |
    Каждый параметр конструктора цвета принимает число.  
    Но сам цвет имеет максимальное значение 255.  
    
    256 - 1 = 255  
    256 - 255 = 1  
    256 - 256 = 0
