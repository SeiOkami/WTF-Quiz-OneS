title: а = +(а = +(а = а))
question: |
    ```bsl
    а = +(а = +(а = а));
    Сообщить(а);
    ```
explanation: |
    Операции можно разбить на шаги:
    - `а = а` => `Истина`
    - `+(а = а)` => `+(Истина)` => `1`
    - `а = +(а = а)` => `a = 1` => `Ложь`
    - `+(а = +(а = а))` => `+(Ложь)` => `0`
options:
    - Сообщит "Нет"
    - Сообщит "0"
    - Вызовет исключение
